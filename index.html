Medical Radiology Summarization

Problem

Radiology reports often contain detailed FINDINGS text and shorter IMPRESSION summaries. This project builds a model that generates the IMPRESSION from the FINDINGS.

Approach

The workflow parses XML reports, cleans and structures the dataset, creates train/test splits, evaluates a baseline, fine-tunes FLAN-T5-small, and evaluates the final model using ROUGE.

Key Features
  • XML report parsing and data preparation.
  • FLAN-T5 fine-tuning for text summarization.
  • ROUGE-1, ROUGE-2, and ROUGE-L evaluation.
  • Reproducible script-based pipeline.
  • FastAPI backend, HTML interface, Dockerfile, and Hugging Face deployment.
Result

The final model achieved a ROUGE-1 F1 score of 0.6244 on the test set, exceeding the required project threshold.

Limitations

A later duplication analysis found train-test content overlap, so future work would deduplicate before splitting and compare the more conservative results.

Visual Product Recommendations

Problem

Users need a simple way to find visually similar clothing products from an uploaded image.

Approach

The system uses CLIP as a visual feature extractor, converts images into normalized embeddings, compares them with cosine similarity, and applies filtering to avoid near-identical recommendations.

Key Features
  • Image indexing, validation, and duplicate removal.
  • CLIP image embedding generation.
  • Cosine similarity recommendation search.
  • Near-duplicate and diversity-aware filtering.
  • Gradio interface, Hugging Face deployment, and optional FastAPI endpoint.
Result

The final documented recommendation output returned five visually similar products with cosine similarity scores above the project guidance threshold.

Limitations

The prototype uses visual similarity only. Future improvements could add metadata-aware filtering, text-to-image search, user feedback, or FAISS for larger catalogues.
